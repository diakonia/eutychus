EUTYCHUS SUPPORT FILES README
=============================

The files in this folder are required in order for Eutychus to function:

audacity.cfg.snip
	Make sure aduacity is closed. Edit and add the above file to the end of
	your audacity.cfg file. On windows it is here:
	
	C:\Users\YourUserName\AppData\Roaming\Audacity\Audacity.cfg

template.aup
	Copy this to your ServiceRecordings folder as set in Audacity.cfg above
